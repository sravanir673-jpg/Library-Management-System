# Library Management System

## Project Description
This is a Python Full Stack Library Management System developed using Django. The application helps manage books, issue books, and maintain library records.

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- JavaScript
- SQLite

## Features
- Add Books
- View Books
- Issue Books
- Return Books
- Search Books
